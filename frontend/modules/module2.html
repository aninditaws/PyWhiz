Conditional Statements (if-else)

In Python, conditional statements (if, else, and elif) are used to execute certain blocks of code based on a condition.

Introduction to Conditional Statements

  • The if statement in Python checks if a condition is true.
  • else provides an alternative if the condition is false.
  • elif allows multiple conditions to be checked in sequence.

    • Basic Syntax for if-else

    age = 18

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

    Nested if Statements

    You can nest multiple if statements to perform more complex checks.

     if age >= 18:
    if age > 65:
        print("You are a senior citizen.")
    else:
        print("You are an adult.")
else:
print("You are a minor.")

    Using Logical Operators

    Logical operators like and, or, and not are used to combine multiple conditions.

    if age >= 18 and age < 65:
    print("You are an adult.")

    Comparison Operators

    ==, !=, >, <, >=, <= are used for comparisons.
