If-Else in Python

If-else statements allow you to control the flow of a program based on conditions.

Example 1: Simple If-Else

age = 18
if age >= 18:
    print("Adult")
else:
    print("Not an adult")

Example 2: Multiple Conditions

score = 85
if score >= 90:
    print("A grade")
elif score >= 80:
    print("B grade")
else:
    print("C grade")
